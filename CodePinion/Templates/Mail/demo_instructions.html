To Report Any Bug Encountered, Follow The Instructions Below!
 

1. Click New Bug Button.
 
 

To activate the Report Bug Form, click the "New Bug" button. This will automatically capture a screenshot of the current code editor window and attach it to the form.
 

2. Fill The Report Bug Form.
 
  1. Provide a Bug Title:

    Enter a concise and descriptive title that accurately summarizes the bug you're reporting.

  2. Verify Screenshot Accuracy:

    Review the captured screenshot to ensure it accurately represents the code editor window at the time of the bug occurrence. If the screenshot is incorrect, click the "Recapture Screenshot" button to retake it.

  3. Elaborate on the Bug:

    Provide a detailed explanation of the bug, including the steps to reproduce the issue, the expected behavior, and the observed behavior. Use the rich text editor to format your description effectively.

  4. Preview Your Changes:

    Before submitting the bug report, click the "Preview" button to review your description and ensure it accurately conveys the issue.

  5. Discard or Submit:

    To cancel the bug report and discard all entered information, click the "Discard" button. To submit the bug report for review and investigation, click the "Report" button.
 

3. Check Bug Menu.
 
 
  1. Pending Bugs:

    Click on "Pending" to view all of the bugs that have not been resolved by out team.

  2. Fixed Bugs:

    Click on "Fixed" to view all the bugs that have been resolved by our team

  3. Toggle View:

    Both the "Pending" and "Fixed" buttons will toggle the "View Bugs" section, allowing you to filter and view bugs according to your preferences.
 
 

4. Access Bug Details.
 
 

Access detailed information about the bug, including a description and the associated screenshot. If you reported the bug, you'll have the option to edit it by clicking the "Edit" button.
